These javascript calendars require the React javascript library. See the main examples page for installation and usage instructions.

The following examples will only show the JavaScript React.createElement or JSX element tag that renders the calendar shown.
For example, the following code will render the default Gondor Calendar shown below, when substituted in the React.createElement(...) line of the main examples page: 
            
    React.createElement(TolkienCalendars.GondorCalendar, null)
The following JSX element tag will render the default Gondor Calendar (shown below) in a React app: 
            
    <GondorCalendar />
Gondor Calendar with only default options: Stewards' Reckoning, Quenya language, today's date, vertical month view, no caption title, and no CSS class to style the table.
