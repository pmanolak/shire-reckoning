These javascript calendars require the React javascript library. See the main examples page for installation and usage instructions.

The following examples will only show the JavaScript React.createElement or JSX element tag that renders the calendar shown.
For example, the following code will render the default Númenor Calendar shown below, when substituted in the React.createElement(...) line of the main examples page: 
            
    React.createElement(TolkienCalendars.NumenorCalendar, null)
The following JSX element tag will render the default Númenor Calendar (shown below) in a React app: 
            
    <NumenorCalendar />
Númenor Calendar with only default options: Kings' Reckoning, Quenya language, today's date, vertical month view, controls displayed, no caption title, and no CSS class to style the table.
