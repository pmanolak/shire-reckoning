See the main examples page for installation and usage instructions.

The following examples will only show the JavaScript React.createElement or JSX element tag that renders the calendar shown.
For example, the following code will render the default Rivendell Calendar shown below, when substituted in the React.createElement(...) line of the main examples page: 
            
    React.createElement(TolkienCalendars.RivendellCalendar, null)


The following JSX element tag will render the default Rivendell Calendar (shown below) in a React app: 
            
    <RivendellCalendar />
Rivendell Reckoning with only default options: Quenya language, Traditional Rules starting from March 22nd, today's date, month view, no caption title, and no CSS class to style the table.
