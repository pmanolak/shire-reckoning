These javascript calendars require the React javascript library.

The following example shows how a default Shire Calendar, with the default styles, may be added to an HTML page. Note that React, ReactDOM, and this library's modules are linked from https://unpkg.com. 
            
<!doctype html>
<html>
    <head>
        <meta charset="UTF-8">

        <script type="text/javascript" src="https://unpkg.com/react@15/dist/react.js"></script>
        <script type="text/javascript" src="https://unpkg.com/react-dom@15/dist/react-dom.js"></script>

        <link href="https://unpkg.com/shire-reckoning/lib/TolkienCalendars.css" rel="stylesheet">
        <script type="text/javascript" src="https://unpkg.com/shire-reckoning/lib/TolkienCalendars.js"></script>
    </head>
    <body>
    <div id='shire-calendar' ></div>
    <script type="text/javascript">
        ReactDOM.render(
            React.createElement(TolkienCalendars.ShireCalendar, {className: "shire-calendar"}),
            document.getElementById("shire-calendar")
        );
    </script>
    </body>
</html>


To use this library as an ES6 module, for example in an app created by create-react-app, first install this library as a dependency by adding shire-reckoning to your app's package.json, then run npm install. Then update the default App.js file with the following code, which will render the default Shire, Rivendell, and Numenor Calendars with the included styles: 
            
    import React, { Component } from 'react';

    import { ShireCalendar, RivendellCalendar, NumenorCalendar } from 'shire-reckoning';
    import 'shire-reckoning/lib/TolkienCalendars.css';

    import './App.css';

    class App extends Component {
        render() {
            return (
                <div className="App">
                    <ShireCalendar className="shire-calendar" />
                    <RivendellCalendar className="shire-calendar" />
                    <NumenorCalendar className="shire-calendar" />
                </div>
            );
        }
    }

    export default App;
