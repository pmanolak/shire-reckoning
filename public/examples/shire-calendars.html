These javascript calendars require the React javascript library. See the main examples page for installation and usage instructions.

The following examples will only show the JavaScript React.createElement or JSX element tag that renders the calendar shown.
For example, the following code will render the default Shire Calendar shown below, when substituted in the React.createElement(...) line of the main examples page: 
            
    React.createElement(TolkienCalendars.ShireCalendar, null)
The following JSX element tag will render the default Shire Calendar (shown below) in a React app: 
            
    <ShireCalendar />
Shire Reckoning with only default options: Shire names of months and days, today's date, vertical month view, controls displayed, no caption title, and no CSS class to style the table.
